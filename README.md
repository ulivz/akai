## Install

Install _akai_:

```bash
# Either globally
yarn global add akai
# Or locally (preferred)
yarn add akai --dev
```

or, If you use `npm`:
```bash
npm i akai -g
npm i akai --save-dev
```



## Usage

```js
const akai = require('akai')

akai()
//=> foo
```
